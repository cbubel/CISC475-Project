Instructions

Viewing/Filtering Students

  1. Select "Students" in top navbar
  2. Click on a student's name to reveal more information, such as the student's schedule, grades, and tags
  3. Use the search box to filter by tags

Adding Students

  1. Select "Add Student" from the top navbar
  2. Fill in the student's information using the form
  3. Click the "Submit" button to add the student to the system

Editing Students

  1. Select "Student" from the top navbar
  2. Click on "Edit" to the right of the student's name you want to edit
  3. Change information in the student's form
  4. Click the "Submit" button to save your changes

Viewing/Editing Courses and Sections

  1. Select "Courses" in top navbar
  2. Click "Edit" to the right of the course whose information/section data you want to edit
  3. Change information in the course's form
  4. Click the "Submit" button to save your changes

Adding Courses

  1. Select "Add Course" from the top navbar
  2. Fill in the course's information using the form
  3. To add a course section, click "Add Section"
  4. Click the "Submit" button to add the course to the system
